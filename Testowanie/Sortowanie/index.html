Wybierz algorytm sortowania

Kroki Bubble Sort

  1. Porównuj sąsiadujące elementy
  2. Zamień miejscami, jeśli są w złej kolejności
  3. Powtarzaj aż do końca listy
  4. Po każdej iteracji największy element "wypływa" na koniec
  5. Kontynuuj, aż cała lista będzie posortowana

Kod w Pythonie


1 def bubble_sort(arr):
2     n = len(arr)
3     for i in range(n):
4         for j in range(0, n-i-1):
5             if arr[j] > arr[j+1]:
6                 arr[j], arr[j+1] = arr[j+1], arr[j]
7     return arr

Kroki Insertion Sort

  1. Weź drugi element w liście (indeks 1) jako element wstawiany
  2. Porównuj go z poprzednimi elementami i przesuwaj je w prawo
  3. Wstaw element na właściwe miejsce
  4. Przejdź do kolejnego elementu i powtórz

Kod w Pythonie


1 def insertion_sort(arr):
2     for i in range(1, len(arr)):
3         key = arr[i]
4         j = i-1
5         while j >= 0 and key < arr[j]:
6             arr[j+1] = arr[j]
7             j -= 1
8         arr[j+1] = key
9     return arr

Kroki Selection Sort

  1. Znajdź najmniejszy element w nieposortowanej części
  2. Zamień go z pierwszym elementem tej części
  3. Rozszerz obszar posortowany o jeden element
  4. Powtarzaj, aż cała lista będzie posortowana

Kod w Pythonie


1 def selection_sort(arr):
2     for i in range(len(arr)):
3         min_idx = i
4         for j in range(i+1, len(arr)):
5             if arr[j] < arr[min_idx]:
6                 min_idx = j
7         arr[i], arr[min_idx] = arr[min_idx], arr[i]
8     return arr

Kroki Merge Sort

  1. Podziel listę na dwie połowy
  2. Rekurencyjnie sortuj każdą połowę
  3. Scal posortowane połowy

2     if len(arr) > 1:
3         mid = len(arr)//2
4         L = arr[:mid]
5         R = arr[mid:]
6         
7         merge_sort(L)
8         merge_sort(R)

Kroki Quick Sort

  1. Wybierz pivot (np. ostatni element)
  2. Podziel listę na elementy mniejsze i większe od pivota
  3. Rekurencyjnie sortuj obie podlisty

Kod w Pythonie


1 def quick_sort(arr):
2     if len(arr) <= 1:
3         return arr
4     pivot = arr[-1]
5     left = [x for x in arr[:-1] if x <= pivot]
6     right = [x for x in arr[:-1] if x > pivot]
7     return quick_sort(left) + [pivot] + quick_sort(right)
